## ­ЪЊЦ Installation Guide

You can install these plugins in two ways:

### ­Ъћ╣ Option 1: Add as a Custom Repository in QGIS

1. Open QGIS Рєњ `Plugins` Рєњ `Manage and Install Plugins`
2. Click on `Settings` tab Рєњ `Add`
3. Name: `GIS GEORGIA`
4. URL: `https://plugins.qgis.ge/plugins.xml`
5. Click `OK` Рєњ Enable and install desired plugin

### ­Ъћ╣ Option 2: Manual Installation

1. Clone or download this repository:
```bash
   git clone https://github.com/GIS-GEORGIA/qgis-plugins-repo.git
```
2. Copy desired plugin folder to QGIS plugins directory:
   - **Windows**: `%APPDATA%\QGIS\QGIS3\profiles\default\python\plugins\`
   - **Linux**: `~/.local/share/QGIS/QGIS3/profiles/default/python/plugins/`
   - **macOS**: `~/Library/Application Support/QGIS/QGIS3/profiles/default/python/plugins/`
